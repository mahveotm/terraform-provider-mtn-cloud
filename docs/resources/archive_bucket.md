# mtncloud_archive_bucket (Resource)

Manages an MTN Cloud archive bucket backed by a storage provider.

## Example Usage

```terraform
# An archive bucket is backed by an existing storage bucket (the storage provider).
resource "mtncloud_archive_bucket" "vault" {
  name             = "project-vault"
  storage_provider = "my-s3-store"
  description      = "Long-term archive for project artifacts"
  visibility       = "private"
}
```

## Schema

### Required

- `name` (String) Globally unique archive bucket name.
- `storage_provider` (String) Storage bucket name that backs this archive bucket. Changing it forces a new archive bucket.

### Optional

- `account_id` (Number) ID of the account that owns the archive bucket.
- `description` (String) Human-readable description of the archive bucket.
- `is_public` (Boolean) Whether the archive bucket is publicly accessible.
- `visibility` (String) Archive bucket visibility: `private` or `public`.

### Read-Only

- `code` (String) Code of the archive bucket.
- `file_count` (Number) Number of files stored in the archive bucket.
- `id` (String) Numeric identifier of the archive bucket.
- `raw_size` (Number) Total raw size of the archive bucket's contents, in bytes.
