# mtncloud_instance (Resource)

Manages an MTN Cloud compute instance using human-friendly provisioning inputs.

## Example Usage

```terraform
# Minimal instance: group, resource_pool, and security_group ("default") are
# inherited from the provider / defaulted. labels/tags merge with provider defaults.
resource "mtncloud_instance" "web" {
  name = "web-01"
  type = "MTN-CS10"
  plan = "G2S4"

  labels = ["web"]          # effective labels_all = ["terraform", "web"]
  tags   = { role = "web" } # tags_all merges provider default_tags
}
```

## Schema

### Required

- `name` (String) Name of the instance. Changing it forces a new instance.
- `plan` (String) Service plan name/code (e.g. `G2S4`) sizing the instance.
- `type` (String) Instance type code (e.g. `MTN-CS10`). Changing it forces a new instance.

### Optional

- `availability_zone` (String) Availability zone. Defaults to the provider's `availability_zone`. Changing it forces a new instance.
- `create_backup` (Boolean) Whether to enable backups at create time. Changing it forces a new instance.
- `create_user` (Boolean) Whether to create a cloud-init user on provisioning. Defaults to `true`. Changing it forces a new instance.
- `description` (String) Human-readable description of the instance.
- `environment` (String) Environment the instance belongs to (e.g. `production`).
- `expire_days` (Number) Number of days after which the instance automatically expires. Changing it forces a new instance.
- `group` (String) Group/site name. Defaults to the provider's `group`. Changing it forces a new instance.
- `labels` (List of String) Labels applied to the instance. Merged with the provider's default_labels into `labels_all`.
- `os_external_network_id` (String) ID of the external (OpenStack) network to attach. Changing it forces a new instance.
- `resource_pool` (String) Resource pool name/code. Defaults to the provider's `resource_pool`; if neither is set and the group has exactly one pool, that pool is used. Changing it forces a new instance.
- `security_group` (String) Primary security group name. Defaults to `default`.
- `security_groups` (List of String) Additional security group names to attach to the instance.
- `shutdown_days` (Number) Number of days after which the instance is automatically shut down. Changing it forces a new instance.
- `tags` (Map of String) Tags applied to the instance. Merged with the provider's default_tags into `tags_all`.
- `timeouts` (Block, Optional) (see [below for nested schema](#nestedblock--timeouts))
- `wait_for_ready` (Boolean) Wait for the instance to reach a running state before completing. Defaults to `true`.
- `workflow_id` (Number) ID of a provisioning workflow to run at create time. Changing it forces a new instance.

### Read-Only

- `group_id` (Number) Resolved numeric ID of the group.
- `id` (String) Numeric identifier of the instance.
- `labels_all` (List of String) Effective labels: the provider's default_labels merged (union) with `labels`.

- `status` (String) Current status of the instance.
